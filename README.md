# Ethernal-Madness 

###How to play a dev version:
- Download the zip file
- Extract the zip file into a folder on your desktop
- open "game.exe"


For every new (Dev)version where the map got updated you should create a new game (Changelog will say to do so)


###How to report a bug:
Go to [Issues](https://github.com/dylanpiera/Ethernal-Madness/issues) and create a new issue. Announce the bugg in as much detail as you possibly can. I'll respond as soon as possible with a possible fix and what not.



###How to upload a file to the project
Everyone in this project has the rights to PUSH. <b>But do NOT push tot he master</b>
Fork the project to your own account (Press "Fork" Next to "Star/Unstar" in the top right of the page)
Change what you want to change. (Please add all files to the "Offproject-Files" folder) then add a Pull Request 
(Go to [Pull Requests](https://github.com/dylanpiera/Ethernal-Madness/pulls)) Please describe what you changed. I'll accept the Pull request if its valid. If change is needed I'll edit it and then push it to the master.
